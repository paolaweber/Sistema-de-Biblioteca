 Sistema de Controle de Biblioteca
Sistema desenvolvido como projeto da disciplina de Algoritmos e Programação.

 Sobre o Projeto
Este sistema permite o cadastro e gerenciamento completo de livros, oferecendo uma interface simples e funcional para controle de acervo bibliotecário.

 Funcionalidades


Função	Descrição
Cadastrar livro	Adiciona um novo livro ao acervo
Consultar livro	Busca informações de um livro específico
Alterar dados	Atualiza informações de livros cadastrados
Remover livro	Exclui um livro do sistema
Listar todos	Exibe todos os livros do acervo
Realizar empréstimo	Registra o empréstimo de um livro
Sair	Encerra o programa
