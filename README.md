Sistema de Controle da Biblioteca
O sistema foi desenvolvido em anexo a disciplina de Algorítmos e programação.

Ele funciona através do cadastro e gerenciamento de livros, e apresenta as seguintes funcionalidades:


Cadastrar livro
Consultar livro
Alterar dados
Remover livro
Listar todos
Realizar o empréstimo
Realizar
Sair
